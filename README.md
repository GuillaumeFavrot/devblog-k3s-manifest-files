# Kubernetes configs files for my devblog app

The secret and configmap files have been excluded from this repository and need to be created with the following variables 

## Configmap

+ ALLOWED_HOSTS
+ ALLOWED_ORIGINS
+ DEBUG
+ DATABASE
+ POSTGRES_DB
+ SQL_HOST
+ SQL_PORT
+ DATABASE_URL

## Secret

+ POSTGRES_USER
+ POSTGRES_PASSWORD
+ DJANGO_SECRET_KEY
